== Description ==

Este plugin é a caixa de ferramentas da equipe VVerner que otimiza e padroniza os projetos entregues pela empresa, além disso adiciona novas funcionalidades para configurações de upload de imagens, SMTP (envio de e-mails) e controle de debug.

== Frequently Asked Questions ==

= Porque este plugin foi instalado? =

A equipe da VVerner utiliza este plugin como base de desenvolvimento de funções personalizados no site.

= Posso excluir este plugin? =

Sim, porém você deverá confirmar a desativação dos plugins e tema dependente dele (se houver)

== Installation ==

= Instalação via arquivo .zip =

* Na página de conexão do painel, baixe o arquivo ZIP do plugin.
* Acesse o painel administrativo do WordPress e vá para "Plugins".
* Clique em "Adicionar Novo" e, em seguida, em "Fazer Upload do Plugin".
* Selecione o arquivo ZIP baixado e clique em "Instalar Agora".
* Ative o plugin após a conclusão da instalação.

= Instalação via WordPress =

* Acesse o painel administrativo do WordPress e vá para "Plugins".
* Clique em "Adicionar Novo" e, em seguida, pesquise por VVerner - Toolbox
* Clique em "Instalar" e depois "Ativar"

= Requisitos =

* PHP com versão 8.0 ou mais recente. PHP 8.3 é a versão recomendada atualmente.
* MySQL com versão 5.6 ou mais recente, OU MariaDB com versão 10.1 ou mais recente
